# Mantine Vite Template
## Features
`master` branch
- Mantine + Vite
- Jest with react testing library
- ESLint

### Testing scripts

- `typecheck` – checks TypeScript types
- `lint` – runs ESLint
- `jest` – runs jest tests
- `jest:watch` – starts jest watch
- `test` – runs `jest`, `lint` and `typecheck` scripts
